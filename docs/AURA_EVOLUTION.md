# Aura — estrutura de UI

## Pastas principais

- `components/ambient/` — fundo animado
- `components/ui/` — SoftCard, GlassCard
- `components/friends/` — lista de amigos, sugestões, modal de perfil
- `components/chat/activities/` — enquetes, quiz, recados no chat
- `components/dashboard/` — shell (servidores, canais, chat, DMs)

## Fluxo (estilo Discord)

- **Servidores** → **Canais** → **Chat de texto**
- **Amigos** (aba Lista) → online / offline / sugestões + **DMs**
- Atividades opcionais no chat: enquete, quiz leve, recado fixado
